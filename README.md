## Mui
===
MUI网页界面集成框架是一套完整的BS模式系统界面解决方案，提供给设计和开发WEB应用的人员使用。
适合用于自动化办公、电子政务和RIA应用等系统的开发中,也可以作为WEB系统后台管理的界面。

[在线预览](http://www.mmcai.cn/mui/)
